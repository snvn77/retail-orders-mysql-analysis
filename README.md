# Retail Orders ETL + MySQL Analysis (Kaggle → Python → MySQL)

Small end-to-end data analysis project:
1) Download dataset from Kaggle using the Kaggle API  
2) Clean data in Python (Jupyter Notebook)  
3) Load into an existing MySQL table  
4) Answer 5 business questions using SQL queries

## Tech stack
- Python (Pandas)
- Jupyter Notebook
- Kaggle API
- MySQL 8.x
- SQLAlchemy + mysql-connector-python

## Project structure
```

```
